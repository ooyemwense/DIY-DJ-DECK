A real-time digital audio effects platform built on the STM32 Nucleo F303RE, designed as an exploration of low-level DSP on resource-constrained hardware. The system samples mono audio at 16 kHz through a timer-driven interrupt service routine, applies a Schroeder reverberation algorithm — implemented as four parallel comb filters feeding two series allpass stages — and outputs the processed signal through the onboard DAC. Effect state is toggled at runtime via a debounced GPIO interrupt, allowing live control without disrupting the audio processing loop. The full signal chain runs in fixed-point arithmetic to stay within the ISR's timing budget, and a ring-buffer architecture between the ADC, DMA, and processing stages ensures sample-accurate throughput without dropouts. Built in PlatformIO using the Arduino-STM32 framework.
